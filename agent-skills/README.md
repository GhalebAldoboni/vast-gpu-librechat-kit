# Agent Skills

This folder contains a reusable skill for AI coding assistants that support `SKILL.md` style workflows.

## Included Skill

- `vast-gpu-librechat` - operational guidance for connecting LibreChat to a Vast.ai GPU running `llama.cpp` through an SSH tunnel.

## Install

Copy `agent-skills/vast-gpu-librechat` into your assistant's local skills directory, then ask the assistant to use the `vast-gpu-librechat` skill when working on this setup.
